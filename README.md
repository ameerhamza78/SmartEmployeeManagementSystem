📄 SEMS – Smart Employee Management System

A modern, advanced GUI-based desktop application for employee management, built with C# Windows Forms.
SEMS provides efficient employee record management, analytics, soft delete with recovery, export features, and a professional UI/UX.

🚀 Features
Add, Update, Delete, and View Employees
Advanced Search & Filters (Name, Department)
Dashboard Analytics (Total, Active, Inactive)
Profile Picture Upload
Export to Excel (.xlsx)
Soft Delete & Restore (Recycle Bin)
Theme Switcher (Dark/Light Mode)
Notification System
Role-based Login & Authorization
Joining Date Tracking
🛠️ Technologies Used
Language: C#
Framework: .NET (Windows Forms)
IDE: Visual Studio 2022
Storage: Text File & File System
