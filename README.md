# Smart Employee Management System (SEMS)

A modern, advanced GUI-based desktop application for employee management, built with **C# Windows Forms**. SEMS provides efficient employee record management, analytics, soft delete with recovery, export features, and a professional UI/UX.

---

## Features

- **Add, Update, Delete, and View Employees**
- **Advanced Search & Filters** (by Name, Department)
- **Dashboard Analytics** (Total, Active, Inactive employees)
- **Profile Picture Upload**
- **Export to Excel (.xlsx)**
- **Soft Delete & Restore (Recycle Bin)**
- **Theme Switcher (Dark/Light Mode)**
- **Notification System**
- **Role-based Login & Authorization**
- **Joining Date Tracking**

---

## Technologies Used

- **Language:** C#
- **Framework:** .NET (Windows Forms)
- **IDE:** Visual Studio 2022
- **Storage:** Text File (for data), File System (for images)

---

## Usage

- **Add Employee:** Fill the form and click "Add".
- **Update/Delete:** Select an employee and use the respective buttons.
- **Recycle Bin:** Restore or permanently delete employees.
- **Export:** Use the export button to save data as Excel.
- **Theme Switch:** Toggle between light and dark modes.
